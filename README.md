## How to run the simulation
1) Run the Classic or the Adaptive CSMA-CA by setting the GUI flag to True in the end.
2) To see different simulations, change the seed at the beginning.

## How to use to plot statistics
1) Run the Classic and the Adaptive CSMA-CA files with the same parameters.
2) Open the statistics file and change the title and output image name according to the parameters used.
3) Run the statistics file with plot total results.
